# CRUD_example
Example CRUD made in Node.js and express for Viaro networks interview 

Instalation:
 
 npm -i 

Build app:

 npm run build

Run server:
 
 npm run serve

API calls on, remember :id is a mongo OID:
    localhost:3000/PRUEBA/users
    "to Create" + /new
    "to Read"   + / or /:id
    "to Update" + /edit
    "to Delete" + /delete/:id
